Phone CDR Analyzer — README                     

PROJECT: Phone CDR (Call Detail Records) GUI Analyzer
LANGUAGE: Python 3
LIBRARIES: tkinter, matplotlib, pandas



 HOW TO INSTALL & RUN


STEP 1 — Install Python (if not installed):
    Download from: https://www.python.org/downloads/
     Tick "Add Python to PATH" during install

STEP 2 — Install required libraries:
    Open Command Prompt / Terminal and type:
        pip install matplotlib pandas

STEP 3 — Run the application:
    Double-click  phone_cdr_analyzer.py
    OR open terminal in the folder and type:
        python phone_cdr_analyzer.py



 LOAD CSV   → Load your own CDR CSV file
SAMPLE DATA → Load 150 demo records instantly
 EXPORT CSV → Save filtered results to CSV
